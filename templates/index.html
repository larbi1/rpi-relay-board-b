Waveshare RPi Relay Board (B)

Raspberry Pi 5  ·  8-Channel Relay Controller

Connecting…
REST API GET /api/status → all relay states (JSON) POST /api/relay/{1-8}/on → turn relay ON POST /api/relay/{1-8}/off → turn relay OFF POST /api/relay/{1-8}/toggle → toggle relay POST /api/relay/{1-8}/pulse?ms=500 → pulse ON for N ms POST /api/all/on → all relays ON POST /api/all/off → all relays OFF (reset)
